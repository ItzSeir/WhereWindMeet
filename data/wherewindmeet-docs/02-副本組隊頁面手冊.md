# 副本組隊頁面手冊

<div class="warning">
<strong>注意：</strong>
仍在整合中，資訊仍然未完整。
</div>

<div class="info">
<strong>介紹：</strong><br>
副本組隊是一個提供玩家<strong>快速建立、尋找及加入副本隊伍</strong>的平台，讓大家不用再花大量時間於世界頻道喊人，只需幾秒即可找到適合自己的隊伍。

目前支援普通俠境、百業俠境、十人團本、五人本、眾志凌霄（爬塔）等各種玩法，並可依照日期、時間及隊伍類型快速尋找適合自己的團隊。
</div>

---

## 功能特色

| 功能 | 說明 |
|------|------|
| 日期分類 | 每天隊伍依照日期整理，方便查看。 |
| 快速組隊 | 玩家可自行建立隊伍或加入其他隊伍。 |
| 多種副本 | 支援的五人本與十人團（普通團、天賦團）及爬塔玩法。 |
| 職業配置 | 可選擇坦克、治療、輸出等定位。 |
| 即時查看 | 一眼即可知道目前隊伍是否還有空位。 |
| 支援手機 | 手機、平板及電腦皆可使用。 |

## 使用建議
- 若臨時無法參加，請取消報名并在Discord副本組隊頻道TAG隊伍第一位玩家説一聲。
- 報名後請留意遊戲好友邀請及私訊通知。
- 此網頁推薦提早確定時間的友友們使用，臨期開新隊伍可能會被自動發文的指令遺忘導致沒人注意。


## 常見問題
<details>
  <summary>Q：需要登入才能報名嗎？</summary>
  <p>不用，只需填寫角色名稱即可快速報名。</p>
</details>

<details>
  <summary>Q：可以建立自己的隊伍嗎？</summary>
  <p>可以，只要點擊 **新增隊伍** 即可建立自己的副本團。</p>
</details>

<details>
  <summary>Q：可以同時報名多個隊伍嗎？</summary>
  <p>可以，但建議確認時間不衝突，以免影響其他玩家。</p>
</details>

<details>
  <summary>Q：手機可以使用嗎？</summary>
  <p>可以，本網站已支援手機、平板及電腦瀏覽。</p>
</details>

<div class="info">
