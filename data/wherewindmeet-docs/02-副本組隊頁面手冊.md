<div class="info">
<strong>介紹：</strong><br>
副本組隊是一個提供玩家<strong>快速建立、尋找及加入副本隊伍</strong>的平台，讓大家不用再花大量時間於世界頻道喊人，只需幾秒即可找到適合自己的隊伍。

目前支援普通俠境、百業俠境、十人團本、五人本、眾志凌霄（爬塔）等各種玩法，並可依照日期、時間及隊伍類型快速尋找適合自己的團隊。
</div>

---

## 功能特色

| 功能 | 說明 |
|------|------|
| 日期分類 | 每天隊伍依照日期整理，方便查看。 |
| 快速組隊 | 玩家可自行建立隊伍或加入其他隊伍。 |
| 多種副本 | 支援的五人本與十人團（普通團、天賦團）及爬塔玩法。 |
| 職業配置 | 可選擇坦克、治療、輸出等定位。 |
| 即時查看 | 一眼即可知道目前隊伍是否還有空位。 |
| 支援手機 | 手機、平板及電腦皆可使用。 |

## 💡 使用建議

- 優先加入缺少自己職業的隊伍，可更快完成組隊。
- 若臨時無法參加，請提前退出隊伍，避免影響其他玩家。
- 建議提前準備好裝備、心法及藥品。
- 報名後請留意遊戲好友邀請及私訊通知。

## ❓ 常見問題

### Q：需要登入才能報名嗎？

不用，只需填寫角色名稱即可快速報名。

### Q：可以建立自己的隊伍嗎？

可以，只要點擊 **新增隊伍** 即可建立自己的副本團。

### Q：可以同時報名多個隊伍嗎？

可以，但建議確認時間不衝突，以免影響其他玩家。

### Q：手機可以使用嗎？

可以，本網站已支援手機、平板及電腦瀏覽。
