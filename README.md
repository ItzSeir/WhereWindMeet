# 燕雲十六聲｜夢回花深處

一個為《燕雲十六聲》打造的百業網站。  
主要提供
---
## 🎯 主要功能

### 📅 時段管理
- 顯示未來幾天的俠境時段
- 支援新增自訂日期與時間
- 可刪除自訂時間段
- 自動排序時段

### 👥 隊伍報名
- 支援單人報名
- 支援批量報名
- 支援常用隊伍快速加入
- 記住常用角色名稱

### 🛡️ 職業與配置
- 自訂職業下拉選單
- 顯示輸出 / 承傷 / 治療數量
- 自動檢查隊伍配置
- 隊伍滿員時自動限制報名

### 🎨 視覺狀態提示
- 空隊伍：預設狀態
- 已有成員：藍色高亮
- 隊伍已滿：紅色提示
- 缺少治療時會有配置提醒

### ⚙️ 其他功能
- 刪除玩家前確認提示
- LocalStorage 本地儲存我的資料/常用隊伍資料
- 手機與桌機自適應
- 深色玻璃擬態風格

---

## 🧩 隊伍規則

- 每隊最多 **10 人**
- 每隊至少需要 **2 位治療**
- 當剩餘位置不足且治療數不足時：
  - 系統會限制部分職業選擇
  - 優先保留治療位置

---

## 🖼️ 介面特色

- 深色玻璃感設計
- 卡片式俠境排版
- 一日一大卡設計
- 清晰的職業統計資訊
- 高對比配色提升閱讀性
- 適合 Discord / 公會 / 固定團 使用
