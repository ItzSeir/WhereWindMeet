# 副本組隊頁面手冊

:::warning 注意

此功能目前仍在持續整合中，部分資訊及功能說明可能尚未完整。

:::

:::info 功能介紹

副本組隊頁面是一個讓玩家能夠 **快速建立、尋找及加入副本隊伍** 的平台。

玩家不需要再花費大量時間於世界頻道招募隊友，只需幾個步驟，即可尋找適合自己的隊伍或自行建立隊伍。

目前支援的玩法包括：

- 普通俠境
- 百業俠境
- 十人團本
- 五人副本
- 眾志凌霄（爬塔）

玩家也可以依照日期、時間及副本類型，快速尋找適合自己的團隊。

:::

---

## 功能特色

| 功能 | 說明 |
|---|---|
| 日期分類 | 隊伍會依照日期整理，方便玩家查看不同日期的隊伍。 |
| 快速組隊 | 玩家可以自行建立隊伍，或加入其他玩家建立的隊伍。 |
| 多種副本 | 支援五人副本、十人團本、普通團、天賦團及爬塔玩法。 |
| 職業配置 | 可選擇坦克、治療及輸出等職業定位。 |
| 即時查看 | 可以快速查看隊伍目前的剩餘位置及報名情況。 |
| 多裝置支援 | 手機、平板及電腦皆可正常使用。 |

## 使用教學

### 功能介紹

#### 1. 創建隊伍

:::steps

@step 創建隊伍

點擊頁面中任意一個 **創建隊伍** 按鈕。

![創建隊伍](data/wherewindmeet-docs/2/5.png)

@step 設定隊伍

選擇以下隊伍資料：

- 日期
- 時間
- 副本類型
- 隊伍名稱
- 其他相關設定

![設定隊伍](data/wherewindmeet-docs/2/6.png)

@step 加入隊伍

完成隊伍建立後，隊長仍需要點擊 **加入隊伍**，將自己加入剛建立的隊伍。

加入時可以使用：

- 快速報名
- 批量報名

:::tip 延伸教學

可以前往以下章節，了解相關功能：

- [2. 快速報名功能](#2-快速報名功能)
- [3. 常用批量隊伍資料功能（或批量報名）](#3-常用批量隊伍資料功能或批量報名)

:::

![加入隊伍](data/wherewindmeet-docs/2/7.png)

:::

---

#### 2. 快速報名功能

:::steps

@step 設定我的資料

點擊頁面左上角的 **我的資料**。

輸入自己的角色資料後，點擊儲存。

![設定我的資料](data/wherewindmeet-docs/2/2.png)

@step 使用快速報名

在加入隊伍時，點擊 **快速報名**，系統便會自動填入已儲存的角色資料。

:::

:::tip 使用建議

建議先完成「我的資料」設定，之後加入不同隊伍時便不需要重複輸入角色名稱及職業。

:::

---

#### 3. 常用批量隊伍資料功能（或批量報名）

:::steps

@step 建立常用批量資料

點擊頁面左上角的 **常用批量隊伍資料**。

輸入以下資料後儲存：

- 隊伍名稱
- 所有成員名稱
- 每位成員的職業
- 其他隊伍資料

![常用批量隊伍資料](data/wherewindmeet-docs/2/3.png)

@step 使用批量報名

在加入隊伍時，點擊 **批量報名**。

系統便會自動填入已儲存的隊伍名稱、成員及職業資料。

:::

:::warning 資料儲存聲明

所有資料只會儲存在目前使用的瀏覽器中。

以下情況發生後，可能需要重新設定：

- 更換瀏覽器
- 清除瀏覽器資料
- 使用其他電腦或手機
- 使用無痕瀏覽模式
- 瀏覽器主動清除網站儲存資料

網站不會自動將這些資料同步至其他設備。

:::

![資料儲存聲明](data/wherewindmeet-docs/2/4.png)

#### 4. 編輯隊伍設定、添加公告及更改隊伍狀態

隊伍建立後，隊長可以編輯隊伍的相關設定。

可調整的內容包括：

- 隊伍名稱
- 日期
- 集合時間
- 副本類型
- 隊伍公告
- 隊伍狀態
- 招募需求

:::info 隊伍狀態

建議隊長依照實際情況更新隊伍狀態，例如：

- 招募中
- 已滿員
- 暫停招募
- 已完成
- 已取消

這樣可以避免其他玩家誤以為隊伍仍然開放報名。

:::

---

#### 5. 隊友標籤設定

隊長可以為隊伍成員加入標籤，方便辨認不同成員的狀態或職責。

例如：

- 主坦
- 副坦
- 主治療
- 輸出
- 指揮
- 待確認
- 臨時成員

:::tip 使用建議

標籤應保持簡短，讓其他玩家可以快速了解成員的定位。

:::

---

#### 6. 更改自身職業

如果報名後需要更改自己的職業，可以在隊伍成員資料中重新選擇職業。

更改完成後，請確認隊伍目前的職業配置是否仍然符合需求。

:::warning 注意

如果更改職業會影響隊伍配置，建議先通知隊長，避免出現坦克、治療或輸出位置不足的情況。

:::

---

#### 7. 取消報名

如果臨時無法參加，請盡早取消報名。

取消後，原本的位置會重新開放給其他玩家加入。

:::important 取消報名後

除了在網頁取消報名外，也建議前往 Discord 副本組隊頻道，標記隊伍中的第一位玩家或隊長，通知對方你無法參加。

:::

---

## 使用建議

- 若臨時無法參加，請盡早取消報名，並在 Discord 副本組隊頻道通知隊長。
- 報名後請留意遊戲好友邀請、私訊及 Discord 通知。
- 建議在確定時間後再進行報名，避免頻繁更改或取消。
- 此網站較適合提前確定時間的玩家使用。
- 若在副本開始前才建立新隊伍，可能因自動公告時間已過，而較少玩家注意到該隊伍。
- 建立隊伍後，請確認自己已成功加入隊伍。
- 隊長應定期查看成員數量及職業配置。

## 常見問題

:::details Q：需要登入才能報名嗎？

不需要。

玩家只需要輸入角色名稱及相關資料，即可進行報名。

:::

:::details Q：可以建立自己的隊伍嗎？

可以。

點擊頁面中的 **創建隊伍** 按鈕，即可建立自己的副本隊伍。

:::

:::details Q：可以同時報名多個隊伍嗎？

可以。

但請確認不同隊伍的日期及時間不會互相衝突，以免影響隊長及其他玩家。

:::

:::details Q：手機可以使用嗎？

可以。

本網站支援手機、平板及電腦瀏覽。

:::

:::details Q：為甚麼更換設備後，我的資料不見了？

因為「我的資料」及「常用批量隊伍資料」只會儲存在目前使用的瀏覽器中。

如果更換設備、瀏覽器或清除瀏覽器資料，便需要重新設定。

:::

:::details Q：建立隊伍後，為甚麼我沒有出現在成員名單中？

建立隊伍後，隊長仍需要點擊 **加入隊伍**，將自己加入隊伍。

可以使用快速報名或批量報名完成加入。

:::

## 📢 注意事項

:::info 注意事項

- 請準時集合，避免影響其他隊友。
- 若隊伍已滿，可以選擇其他時段或自行建立隊伍。
- 請保持友善交流，共同維持良好的遊戲環境。
- 報名前請確認日期、時間、副本及職業資料。
- 若需要取消報名，請盡早通知隊長。
- 本站僅提供組隊媒合功能，不參與遊戲內交易或玩家糾紛處理。

:::
